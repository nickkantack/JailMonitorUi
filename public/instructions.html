Jail Monitor is a tool that monitors a list of individuals and sends a daily email report indicating which individuals recent appeared to be in custory in the Pennington County Jail and which individuals recently left custody. A report will look something like this:

The tool analyzes the Pennington County Jail website on a daily basis to determine which individuals are in and out of custody.

Setup

Follow these steps to begin receiving daily emails.

  1. Obtain an account password from the Jail Monitor developer (nickkantack@gmail.com).
  2. Create your list of names. There are two options:
    • Use the Jail Monitor web interface.
    • Connect Jail Monitor to an Excel file on your computer.

After these steps are complete, you will start receiving daily emails.

Using the Jail Monitor web interface

You can access the web interface at jailmonitor.penncolabs.org. The very first time you visit the site, it will ask you for your password.

Note: If you incorrectly enter your password, you'll receive an error message and can retry entering your password. If repeated attempts are not successful, contact the developer to have your password reset (nickkantack@gmail.com).

After you successfully enter your password, credentials will be stored on your device and you will not need to enter your password on future visits. you will see the main interface for managing your list of names.

Initially, you will see an empty list of names. You can begin adding names by clicking the Add name button. This will produce a row in the table that will allow you to enter the first and last name of an individual to monitor.

After adding an name you'll notice the banner at the top of the screen indicates there are unsaved changes. This means you've altered the list in your browser but the server has not saved the change. To save your changes to the server, click the Save button in the upper left corner of the page.

Once your changes are saved, the banner at the top of the page indicates so. If you attempt to close the page with unsaved changes, the browser will inform you that you have unsaved changes and give you a chance to return and save.

To remove a name, click the button to the right of the name.

Some notes on the web interface

  • You can enter names with any style of capitalization you like, but the names are stored on the server all lower-case. This is why you'll see names in all lower case anytime the list is loaded from the server, even if you originally entered them with capitalization.
  • If someone has a name that contains multiple separate words (e.g. "red shirt") feel free to enter it with the space. The Jail Monitor will recognize this name even if the jail website takes a slightly different approach (e.g. "red-shirt" or "redshirt").
  • You can enter names in any order. The next time your list of names is loaded from the server, it will display in the UI sorted alphabetically by the first letter in the last name.
  • Once you log in on a device, that device will remain logged in. This means that if someone else uses the same computer to access the Jail Monitor web interface, they will be able to view and manage your list of names. If you need to revoke a device's access to your list of names, or if you want to allow someone else to manage their list of names on the same device, you can use the Log out button at the top-right of the web interface.

Connecting Jail Monitor to an Excel file

It is possible to configure a program on your computer that will regularly upload an Excel file to the Jail Monitor server instead of using the web interface. This is a good option for users who already maintain a spreadsheet with names. To configure this connection, contact the developer (nickkantack@gmail.com).

General help or reporting bugs

If you are experiencing issues with Jail Monitor, you can contact the developer for assistance (nickkantack@gmail.com). You can also suggest changes to the tool.
